dotfiles
========

Mes fichiers de conf basiques.

Installation Linux
------------------

Clôner le dépot Git, puis :

    $ cd dotfiles
    $ bash install.sh

Ensuite, synchroniser le dépot Git au besoin.

Installation Windows
--------------------

Clôner le dépot Git, puis lancer le script `install.bat`.
